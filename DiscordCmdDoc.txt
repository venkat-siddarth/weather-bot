```
+----------------------------------+---------------------------------------------------------------------+
| Commands                         | Description                                                         |
+----------------------------------+---------------------------------------------------------------------+
| $hello                           | Greets you                                                          |
+----------------------------------+---------------------------------------------------------------------+
| $help                            | provides short list of commands                                     |
+----------------------------------+---------------------------------------------------------------------+
| $weather<place-name>             | Returns the recently updated weather data of specified location     |
+----------------------------------+---------------------------------------------------------------------+
| $weather<latitude,longitude>     | Returns the recently updated weather data of mentioned co-ordinates |
|                                  | co-ordinates should have only magnitudes(no degree/direction)       |
+----------------------------------+---------------------------------------------------------------------+
| $save_tags<tagName> <place-name> | The specified location will be saved in the form of tags            |
+----------------------------------+---------------------------------------------------------------------+
| $weather #tagName                | Finds the weather data for the value of <tagName>                   |
+----------------------------------+---------------------------------------------------------------------+
```
